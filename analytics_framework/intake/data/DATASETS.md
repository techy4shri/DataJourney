

# DATASETS

This directory contains the following datasets:

| Dataset                        | Description                                                                                       | Key Columns / Features                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| **YoutubeCommentsDataSet.csv** | This dataset contains YouTube comments along with their associated sentiment labels. The sentiment is categorized as positive, neutral, or negative, providing insights into the tone of user comments. | `Comment`, `Sentiment`                                                                                    |
| **trees.csv**                  | This dataset includes measurements of trees, such as their girth (in inches), height (in feet), and volume (in cubic feet). It is useful for analyzing tree growth patterns and relationships between these variables. | `Index`, `Girth (in)`, `Height (ft)`, `Volume(ft^3)`                                                      |
| **global_bleaching_environmental.csv** | This dataset provides detailed environmental data related to coral bleaching events. It includes information such as site location, ocean name, temperature metrics, bleaching levels, and other environmental factors. It is valuable for studying the impact of environmental changes on coral reefs globally. | `Site_ID`, `Latitude_Degrees`, `Longitude_Degrees`, `Ocean_Name`, `Bleaching_Level`, `Temperature_Kelvin` |
| **TWLO_stocks.csv**            | This dataset contains historical stock data for Twilio (TWLO). It includes daily records of the stock's open, high, low, and close prices, as well as the trading volume. This data is useful for analyzing stock performance and trends over time. | `date`, `open`, `high`, `low`, `close`, `volume`                                                          |
